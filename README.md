# Twentieth-Century Global Power Network
## A Web-Scraped NLP and Network Analysis Project
An Overview:
This project analyzes relationships between countries during major twentieth-century historical events. Using web scraping, natural language processing (NLP), and network analysis, it identifies how countries interacted and which ones played central roles in shaping world history.

What this project does?
- Scrapes a list of countries and Wikipedia pages on 20th-century history
- Uses NLP to extract country mentions and connections
- Builds a relationship graph based on co-occurrences
- Calculates degree, closeness, and betweenness centrality
- Creates an interactive network visualization showing global influence and communities

Tools & Technologies:
- Python, Jupyter Notebook
- BeautifulSoup (web scraping)
- spaCy / NLTK (NLP)
- NetworkX (graph analysis)
- PyVis / Plotly (interactive visuals)

Key Outcomes:
Identified the most influential countries in 20th-century events
Mapped how nations were connected through conflicts, alliances, and global shifts
Created a clean, dynamic network chart suitable for public viewing
